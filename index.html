How to Upload Pictures on Trinket

Follow these simple steps to add pictures to your Trinket projects:

1 Open Trinket and Your Project

Go to trinket.io and open your coding project.

2 Find the File Upload Button

On the left side panel where your files are listed, look for:

  • A plus (+) icon
  • An "Upload" button
  • "Add files" option
  • A folder icon
3 Click the Upload Button

When you click it, a file selection window will appear from your computer.

4 Select Your Picture(s)

Browse your computer to find the image file you want to upload (photos, drawings, etc.).

Select the file and click "Open" or "Choose".

Supported formats: .png, .jpg, .gif (max few MB)
5 See Your Picture in Trinket

Your uploaded image will appear in Trinket's file list.

6 Use the Picture in Your Code

Reference your image file in code. For example, in HTML:

<img src="myphoto.png" alt="My Picture">

Or in Python with Turtle:

screen.addshape("myphoto.png") turtle.shape("myphoto.png")
Extra Tips for Success
  • Use small image files (under 1MB works best)
  • Common formats like PNG and JPG are most reliable
  • Double-check your filename spelling in code
  • Ask for help if you get stuck!

Example Code Using Uploaded Images

HTML example showing an uploaded picture:

<!DOCTYPE html> <html> <head> <title>My Picture</title> </head> <body> <h1>Look at my uploaded picture!</h1> <img src="myphoto.jpg" alt="My awesome photo" width="400"> </body> </html>

Python Turtle example with an uploaded image:

import turtle screen = turtle.Screen() screen.addshape("mydrawing.png") t = turtle.Turtle() t.shape("mydrawing.png") t.penup() t.goto(0, 100)
